# Portfolio Website

A responsive **static portfolio website** built using HTML, CSS, JavaScript, and Bootstrap. The site is designed to present projects, skills, education, and certifications in a clean, structured layout and is deployed using **GitHub Pages**.

---

## Tech Stack

* **HTML5**
* **CSS3**
* **Bootstrap**
* **JavaScript**
* **jQuery**

### Libraries & Plugins

* WOW.js (animations)
* SlickNav (mobile navigation)
* Nivo Lightbox
* MixItUp (portfolio layout)
* CounterUp & Waypoints
* Formspree (contact form integration)

---

## Project Structure

```
├── index.html
├── assets/
│   ├── css/        # Stylesheets
│   ├── js/         # JavaScript & plugins
│   ├── img/        # Images, icons, gallery
│   ├── fonts/      # Font & icon libraries
│   └── favicon/    # Site favicon
```

---

## Features

* Responsive design (desktop & mobile)
* Animated UI components
* Project gallery with external links
* Skills visualization using icons
* Certification image slider
* Functional contact form

---

## Deployment

Hosted on **GitHub Pages**.
Updates are automatically deployed when changes are pushed to the main branch.
